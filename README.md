## About

This repository contains Computer Graphics and Gaming coursework programs written in C. The examples are grouped around a question set and demonstrate drawing, transformations, and graphics fundamentals.

## Repository Map

- `Basic_to_Advance_(34_Questions)/`: C graphics solutions and the source question list.

## Note

Many examples expect a C graphics setup such as Turbo C/C++ or a compatible graphics library environment.


## Getting Started

Clone the repository:

```bash
git clone https://github.com/CoderFatherBB/Computer_Graphics_and_Gaming.git
cd Computer_Graphics_and_Gaming
```

Open the folder or notebook you want to explore. Most directories are self-contained, so start from the local `README.md`, source file, notebook, or project folder for that topic.
